# 📊 StatistikPro - Interactive Statistical Calculator

StatistikPro is a modern web application built with **Next.js** designed to simplify descriptive statistical data analysis. This app instantly calculates central tendency, data dispersion, and analyzes curve shapes (Skewness & Kurtosis), complete with a step-by-step breakdown just like manual calculations on paper.

## ✨ Key Features

- **Dual Data Support:** Computes both Ungrouped Data and Grouped (Interval) Data.
- **Automated Calculations:** Calculates Mean, Median, Mode, Variance, Quartiles, Percentiles, and Standard Deviation.
- **Curve Analysis:** Measures the degree of Skewness and Kurtosis of the data distribution.
- **Dual Calculation Methods:** Compares results using the **Standard Deviation Method** and the **Coding Method ($u$).**
- **Step-by-Step Process:** Provides detailed calculation steps using professional mathematical formatting (LaTeX).
- **Interactive Visualization:** Displays frequency distribution through dynamic Histograms.
- **Import & Export:** Supports `.csv` data uploads and exports the analysis results directly to **PDF**.

## 🛠️ Tech Stack

This project is built using a modern tech stack:
- **Framework:** [Next.js](https://nextjs.org/) (React 19 / App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Math Rendering:** React-LaTeX-Next (KaTeX)
- **Charts:** Chart.js & React-Chartjs-2
- **PDF Export:** React-To-Print
